CSS

You can add to exsiting styles like <h1> by either adding inline styles like <h1 "text-align: center;"> or via using a css style sheet/file normally name "style.css".

Within the css file you can use the id attribute, matching the id="name" with the #name in the css style sheet, this is used for one off reasons, as you can't use id="name" on more then one html markup.

You can use class type that can be used in many different attributes simple by adding the class as the attribute. You can also add the same type of styles to the same mark up by adding a comma between the marks like ul, ol { list-style:none }.

Finally, there's types these start with a "." - for example - .type, they can be used with the class type attribute to modify individual html tags.

Order of specificity

  1. inline
  2. id
  3. class
  4. type

Sass

Sass basically puts variables into css to make it easier to quickly updates styles You can also nest and make models, like inheriting from a base document.
